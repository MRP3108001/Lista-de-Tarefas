Guia de Acesso e Utilização do App — Lista de Tarefas

1. Obter o APK do Aplicativo

Acesse o repositório no GitHub:

https://github.com/MRP3108001/Lista-de-Tarefas

2. Navegue até a branch v2

Vá até a pasta: /APKversao/

3. Baixe o arquivo .apk

4. Instalar o APK no Dispositivo Android, Transfira o arquivo .apk para o seu celular ou emulador (exemplo LDPlayer) por cabo, e-mail ou  nuvem

5. No celular, toque no arquivo para instalar

Se for solicitado, habilite a opção “Instalar apps de fontes desconhecidas”, Após a instalação, abra o app Lista de Tarefas
